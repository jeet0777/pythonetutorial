While statement

Python provides looping with while statement. If you want to execute block of statements repeatedly, depending upon the condition, then you can go for while statement.

Syntax:
while expression:
  Statement(s)
else:
  Statement(s)

Python supports else clause with loop.
It will execute the block of statement repeatedly as long as the expression is true. Here statement may be a single statement or block of statements.
A loop becomes infinite loop if a expression never becomes FALSE. This results in a loop that never ends.

The else clause is optional. If it is used with a while loop, the else statement is executed when the expression becomes false.

When condition becomes false, the loop terminates and if else clause is present then it will executes otherwise first statement after the loop will be executed.

It can skip else clause in following scenarios:
1.If loop is broken out due to some break statement or a return statement(if the loop is in a function or method)
2.If exception occurs, python skips the else clause and looks for a suitable exception handler.


Example:
number = 23
running = True
while running:
  guess = int(raw_input(“Enter a number”))

  if guess == number:
    print “You guessed right”
    running = False
  elif guess < number:
    print “It is higher”
  else:
    print “It is smaller”
else:
  print “end of while loop”

print “Done”

In this example, user is allowed to keep guessing repeatedly until he guesses correctly. Here if statement and raw_input() are inside while loop and running variable is set to True initially. Program will check the if running variable is true and then proceeds to execute while-block. After this block is executed, it will again check the condition (running variable is true or not ). If it is true it will again executes while-block otherwise it will executes the optional else block, if present and continue to the next statement.
