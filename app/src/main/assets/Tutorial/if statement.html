if Statement

There is no switch statement in python you can use if..elif..else statement to do the same thing.

Syntax:
if expression:
  Statements
else:
  Statements

If statement work as follows:
If the expression is true, it will run the block of statement(if block).
Otherwise it will run another block of statement(else block)

Python uses indentation levels to know which statements belongs to which block. If statement contains a colon at the end to indicate python that a block of statements follows.

Example:
if x<0:
  print “x is Negative”
else:
  Print “x is Positive”

In given example, if x is less than 0 then it will run if block and print x is Negative.
Otherwise it will run the else block and print x is Positive.

We can write if statement in single conditional expression.
Syntax:
expression1 if boolean_expression else expression2

If boolean_expression evaluates true, then the result will be expression1 otherwise the result will be expression2.
One common programming pattern is to set variable to a default value, and then change the value to if necessary.For example due to request by user or to the account for the platform on which the program is being run.
Conventional if statement

Offset = 20
If not sys.platform.startswith(“win”):
  Offset = 10

The system.platform holds the name of current platform for example “win32” or “linux2”. The same thing can be achieved using following conditional expression

Offset = 20 if sys.platform.startswith(“win”) else 10

Here In this case there is no need of parenthesis. But parenthesis make things clearer for human readers and to avoid subtle trap.
