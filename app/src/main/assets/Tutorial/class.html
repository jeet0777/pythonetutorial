Class

Classes are a user-defined data type that is used to encapsulate data and associated functions together. It also helps in binding data together into a single unit.

We can think of class as a sketch (prototype) of a house. It contains all the details about the floors, doors, windows etc. Based on these descriptions we build the house. House is the object.

As, many houses can be made from a description, we can create many objects from a class. An object is also called an instance of a class and the process of creating this object is called instantiation.

Defining a Class
A class is created using the class keyword. The fields and methods of the class are listed in an indented block.

class Person:
  pass # An empty block

We create a new class using the class statement and the name of the class. This is followed by an indented block of statements which form the body of the class. In this case, we have an empty block which is indicated using the pass statement.

We will use following code sample to read all class concepts.

class Person:
  '''Common base class for all persons'''
  pCount = 0

  def __init__(self, name, age):
    self.name = name
    self.age = age
    Person.pCount += 1

  def displayPerson(self):
    print "Name : ", self.name, ", Age : ", self.age

  #Access attribute
  print "Persons :", Person.pCount

  #Print DocString
  print "DocString :", Person.__doc__

  #create first object
  p1 = Person("Nick", 20)
  #create second object
  p2 = Person("Rick", 25)
  p1.displayPerson()
  p2.displayPerson()
  print "Total Persons %d" % Person.pCount

OUTPUT :
Persons : 0
DocString : Common base class for all persons
Name : Nick , Age : 20
Name : Rick , Age : 25
Total Persons 2

Accessing Attributes
You access the object's attributes using the dot operator with object. Attributes may be data or functions.
#Access attribute
print (Person.pCount)

There are also special attributes in it that begins with double underscores (__). For example, as given in above example __doc__ gives us the docstring of that class.

Creating an Object
we create an object/instance of this class using the name of the class followed by a pair of parentheses.
>>>ob1 = MyClass()

The first argument of the function in class must be the object itself. This is conventionally called self (You must have noticed displayPerson function). It can be named otherwise but we highly recommend to follow the convention.

Constructors
As we know class functions that begins with double underscore (__) are called special functions as they have special meaning.
__init__ function is also called constructors in Object Oriented Programming (OOP). We use it to initialize all the variables. __init__ gets called whenever a new object of that class is instantiated.
