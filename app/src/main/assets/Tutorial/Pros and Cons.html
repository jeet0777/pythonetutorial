Pros&Cons

Pros of Python
1 - Python is easy to learn for even a novice developer.
2 - Supports multiple systems and platforms.
3 - Object Oriented Programming-driven.
4 - Python has a plethora of frameworks that make web programming very flexible.
5 - Gives rise to quick development by using less code. Even a small team can handle Python effectively.
6 - Allows to scale even the most complex applications with ease.
7 - A large number of resources are available for Python.
8 - Offers a built in testing framework to set debugging time and enable fastest workflows.
9 - IT giants like Yahoo, Google, IBM, NASA, Nokia and Disney prefer Python.

Cons of Python
1 - Python is slow.
2 - Python is not a very good language for mobile development.
3 - Python is not a good choice for memory intensive tasks.
4 - It's near impossible to build a high-graphic 3D game using Python.
5 - Has limitations with database access.
6 - Python is not good for multi-processor/multi-core work.
